What is the idea?

The idea is to implement a simple Shiny interactive application, which would allow for a user to utilize user interface controls to get a plot built for the mtcars dataset which is supplied in R.

The plot was built depending on the following inputs:

  • Y variable
  • X variable
  • split variable
  • smoothing method

Approach to implementation

  1. Transform dataset's variable names to more user-friendly
  2. Separate factor variables, to be use for the split, from all the others
  3. Create a shiny application with user interface (ui.R) and processing logics behind (server.R)
  4. Deploy application on a public portal shinyapp.io
  5. Share source code on github.com page
